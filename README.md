# Basic Keylogger in Python
This is a simple addition to a basic keylogger created by Github user Brick Sigma that records all keyboard input and commits them into a file called data.txt

To run the program you'll first have to install the "keyboard" package using `pip install keyboard`

# [IMPORTANT] Note for anyone trying to use this maliciously
I am not an avid Python user, this is purely for educational purposes within my collegiate research course.

It is not an application intended for serious real-world malicious use, and is easily detectable by its use of the keyboard package. The only functionality it adds over Brick Sigma's original creation is that it marks the file created as hidden. It only works with Windows systems as a result of its shoddy programming.
